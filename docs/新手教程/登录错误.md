## 以 L-、D-、S- 或 P- 开头的市场错误

这些字母错误代码是特定于市场的，表明你在登录时遇到了未知客户端错误。你可以尝试通过以下方法清除错误：

- **检查你的游戏版本**：确保你的游戏是最新版本，并正在运行最新版的 Minecraft。
- **检查账户登录**：确保你的游戏已绑定到 Microsoft
  账户。你可以在主菜单中查看，点击右下角的登录按钮即可确认。有关这方面的帮助，请参阅“[账户与登录](https://help.minecraft.net/hc/en-us/categories/12617305106701)
  ”页面。
- **重启系统**：退出并完全关闭 Minecraft，重启你的设备，然后尝试重新登录。

## 通用错误代码排查

- **更新你的游戏版本**：检查游戏平台的设置，确保你已启用自动更新，并正在运行最新版的 Minecraft。有关手动更新 Minecraft
  的信息，请参阅我们的“[下载与安装](https://help.minecraft.net/hc/en-us/sections/27166490706957)
  ”帮助页面。如果你进行了更新，请务必重启设备和游戏。

- **注销并重启**：[注销 Minecraft](https://help.minecraft.net/hc/en-us/articles/360045828292)、Microsoft Store、Xbox
  应用程序及任何其他可能正在运行的 Minecraft
  服务，并关闭相关程序。然后关闭并重启你的游戏设备，再重新打开 Minecraft 并登录。

- **检查网络连接**：请参阅“[排解 Minecraft 网络连接错误](#排解Minecraft网络连接错误)”页面，获取相关建议。

- **调整视频设置**
  ：低带宽、内存不足或处理能力不足都可能影响游戏性能。请参阅“[优化 Minecraft 视频设置与性能](https://help.minecraft.net/hc/en-us/articles/4410246518669)
  ”页面获取相关技巧。

- **运行 Xbox Gaming Services 修复工具（仅限 PC）**：此工具可帮助你在安装或更新后解决 Minecraft
  或启动器的问题。请前往 [PC版 Gaming Services 修复工具页面](https://support.xbox.com/en-US/help/games-apps/troubleshooting/gaming-services-repair-tool)
  使用它。

- **禁用附加内容**
  ：如果你在世界中运行了大量模组，这些第三方应用可能彼此或与其他系统不兼容。你可以尝试启用或禁用增强内容。有关详细信息，请参阅“[禁用 Minecraft 附加内容](https://help.minecraft.net/hc/en-us/articles/24121220131597-Disabling-Minecraft-Add-Ons)
  ”页面。

- **等待重试**：临时性的服务器故障会在 [PlayFab](https://status.playfab.com/)
  和/或 [Xbox Live](https://support.xbox.com/en-US/xbox-live-status) 状态页面上公告。如果存在服务中断，你需要等待其恢复后再尝试。

## 排解Minecraft网络连接错误

### 游戏设备排查

- **重启系统**：在尝试更高级的排查方法之前，建议你先重启所有硬件与软件设备：

    1. **注销并关闭程序**：
       [注销 Minecraft](https://help.minecraft.net/hc/en-us/articles/360045828292-How-to-Sign-Out-of-Minecraft)
       、Microsoft Store、Xbox 应用程序以及任何可能正在运行的 Minecraft 服务，并关闭它们。

    2. **关闭你的游戏平台**：
       完全关机你的电脑、移动设备或游戏主机。

    3. **重启你的网络连接**：
       关闭路由器和调制解调器，等待 20 秒，让电源彻底断开，再重新开机。

    4. **重启游戏**：
       重新启动你的游戏设备，连接到互联网，重启 Minecraft 并使用 Microsoft 账户登录。

    5. **检查订阅服务**：
       如果你在主机上进行 Minecraft
       多人游戏，必须订阅以下服务之一：[Xbox Game Pass Core](https://www.xbox.com/en-US/xbox-game-pass)、[Nintendo Switch Online](https://ec.nintendo.com/US/en/membership/)
       或
       [PlayStation Plus](https://www.playstation.com/en-us/ps-plus/)，才能在线游玩。

### 网络连接排查

- **检查网络连接**：无法连接服务器或进行在线游戏，通常是由于网络不稳定或断断续续。请重启你的路由器和网络连接，并确认你和主机（房主）都能正常访问其他网站。

- **暂停其他网络活动**：临时停止其他软件的下载、视频串流或占用带宽的任务。

- **关闭其他占用网络的设备或程序**：关闭其他正在进行串流、游戏或大量使用网络的设备或应用程序，看看网络状况是否改善。

- **检查网速**：运行你的网络服务商推荐的网速测试，确认网络连接的强度和速度是否足够。

- **确认无线信号强度**：如果你通过无线网络游玩，尝试将你的游戏设备移近路由器。

- **检查 DNS 设置**：如果你手动更改过设备的 DNS 设置，建议恢复为“自动”。

- **检查 NAT 设置**：确保你的路由器使用“中等”或“开放”类型的 NAT（网络地址转换）设置。

- **检查开放端口**：如果你使用了防火墙、防毒软件或
  VPN，请确认它们没有阻挡或减慢连接速度。需要开放的端口包括：80（不使用有破坏性的代理）、443（不使用有破坏性的代理）、3748、19132、19133

## 在更衣室中装备角色创建器物品与皮肤

你可以随时更改你的 Minecraft 角色外观！以下是如何在更衣室中为角色装备新物品的说明。

**注意**：更衣室功能仅适用于 Minecraft：基岩版（Bedrock Edition）。

### 如何在 Minecraft 中使用角色创建器

**角色创建器物品**

如果你想为角色添加 角色创建器（Character Creator） 的物品，请依照以下步骤操作：

1. 启动 Minecraft：基岩版

2. 选择 更衣室（Dressing Room）

3. 选择你要编辑的角色栏位，或新建一个

4. 选择 编辑角色（Edit Character）

5. 打开 身体（Body） 或 风格（Style） 标签页

6. 在对应分类中找到你要使用的物品

7. 点击 装备（Equip）

### 如何在 Minecraft: Bedrock Edition 中装备经典皮肤

如果你想使用在市集（Marketplace）购买的经典皮肤，请按照下列步骤：

1. 启动 Minecraft：基岩版

2. 选择 更衣室（Dressing Room）

3. 选择你要更换皮肤的角色栏位，或新建一个

4. 选择 编辑角色（Edit Character）

5. 在左侧点击 经典皮肤（Classic Skins）

6. 浏览并选择你想使用的皮肤

7. 点击 装备（Equip）

**注意**：经典皮肤无法与角色创建器物品混搭使用。
