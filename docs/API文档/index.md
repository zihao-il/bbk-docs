## 请求头类型

```json
{
  "Content-Type": "application/x-www-form-urlencoded"
}
```

## API说明

本站共有三个API分别为：

- `/last_version`：获取最新版本数据
- `/get_version`：获取大版本数据
- `/search_version`：搜索版本数据
